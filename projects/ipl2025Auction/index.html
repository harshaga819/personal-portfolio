IPL 2025 Auction Analysis

Description

IPL 2025 Auction Analysis is a comprehensive data analysis project that explores the trends and insights of the IPL (Indian Premier League) 2025 player auction. This project utilizes web scraping techniques to collect player and auction data from ESPN Cricinfo, providing a deep dive into player performance, past auction trends, and price analysis.

The project is built using Python, with libraries like BeautifulSoup for web scraping, Pandas for data cleaning and analysis, and Matplotlib for data visualization. Key data such as player statistics, auction prices, and past performance are scraped and cleaned to ensure accuracy. Interactive dashboards and visualizations are created to help users understand auction dynamics and trends more effectively. The application provides valuable insights on which players are likely to be high-value targets based on their performance, previous auction prices, and market demand.

This project is ideal for IPL teams, analysts, and cricket enthusiasts who are interested in making data-driven decisions during the IPL auction process. It demonstrates the power of combining web scraping with data science techniques to extract meaningful insights from large datasets.

Technology Used

  1. Python

  2. BeautifulSoup

  3. Pandas

  4. Matplotlib

  5. Web Scraping

  6. Data Visualization
