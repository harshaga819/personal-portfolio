Simson Game

Live Demo

Description

Simson is a fun and challenging memory game that keeps players engaged as they replicate color sequences that grow more complex with every level. Built with JavaScript, HTML, and CSS, it reflects my passion for creating intuitive, responsive web applications. The seamless gameplay, polished design, and interactive mechanics showcase my skills in front-end development and attention to detail.

Key Features

  • Dynamic color sequence generation
  • Level-based increasing difficulty
  • Responsive design for mobile and desktop
  • Polished animations and transitions
  • Interactive UI with audio feedback

What I Learned

While developing Simson Game, I improved my skills in JavaScript event handling, animations, and creating responsive layouts. It helped me understand game logic and sequencing better, especially using timers and managing user input effectively.

  2. CSS
