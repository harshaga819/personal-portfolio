Thundra

Description

Thundra is a dynamic weather application built using HTML, CSS, and JavaScript. It features location-based weather reporting, providing real-time weather updates for any area with just a button click. Thundra enhances user experience with its customizable interface, allowing users to change the application's color to suit their preferences. Designed for both functionality and aesthetics, Thundra combines modern web development practices with user-centric design, making it a versatile tool for accessing weather information effortlessly.

Technology Used

  1. HTML

  2. CSS

  3. JavaScript
