Thundra

Live Demo

Description

Thundra is a dynamic weather application built using HTML, CSS, and JavaScript. It features location-based weather reporting, providing real-time updates for any area with just a button click. Thundra enhances user experience with its customizable interface, allowing users to change the app's color to match their preferences. Designed for both function and visual appeal, Thundra blends modern web development with user-first design.

Key Features

  • Location detection for instant weather updates
  • User interface with customizable color schemes
  • Responsive design for all devices

What I Learned

  • Integrating third-party weather APIs
  • Implementing geolocation features in JavaScript
  • Improving UI/UX with responsive layouts

Technologies Used

  1. HTML
  2. CSS
  3. JavaScript
